## Лабораторная работа 2: Калькулятор. JavaScript

### Цель работы
Знакомство с инструментами построения пользовательских интерфейсов web-сайтов: HTML, CSS, JavaScript. Добавление интерактивности к калькулятору.

### Задания
1. Создание калькулятора. Функции на JavaScript.

2. **Дополнительные задания:**
   - Запрограммировать операцию смены знака (+/-)
   - Запрограммировать операцию вычисления процента (%)
   - Добавить кнопку стирания введенной цифры назад (backspace)
   - Сделать смену цвета фона по кнопке
   - Запрограммировать операцию вычисления квадратного корня
   - Запрограммировать операцию возведения в квадрат
   - Запрограммировать операцию вычисления факториала
   - Добавить кнопку, которая за раз добавляет сразу три нуля
   - Запрограммировать накапливаемое сложение
   - Запрограммировать накапливаемое вычитание
   - Сделать смену цвета окна вывода результата по кнопке
   - Добавить в калькулятор индивидуальную операцию
