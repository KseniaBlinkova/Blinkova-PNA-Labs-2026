## Лабораторная работа 1: Калькулятор. HTML/CSS

### Цель работы
Знакомство с инструментами построения пользовательских интерфейсов web-сайтов: HTML и CSS. Создание интерактивного калькулятора.

### Задание
1. **HTML-разметка**
   - Изучение базовой структуры HTML-документа
   - Создание проекта в VS Code
   - Верстка калькулятора с использованием семантических тегов
   - Добавление идентификаторов (id) для всех активных элементов

2. **CSS-стилизация**
   - Применение CSS к HTML-документу
   - Создание селекторов (элемент, id, class)
   - Стилизация верстки калькулятора

3. **Дополнительные задания:**
   - Поменять цветовую палитру калькулятора
   - Сделать фон калькулятора темным (ночная тема)
   - Изменить форму кнопок (квадратные вместо круглых)
   - Сделать окно вывода со скругленными краями
   - Поменять шрифт цифр
   - Сделать шрифт более толстым
   - Изменить цвет при наведении мышки на кнопку на другой
   - Добавить надпись "ЛР выполнена"
   - Выровнять калькулятор по центру
   - Увеличить размер окна вывода
   - Добавить кнопку для смены темы
   - Сделать шрифт тоньше
   - Сменить цвет шрифта
   - Добавить любое изображение на фон
   - Добавить кнопку с ссылкой на GitHub
   - Добавить поле с выпадающим списком
   - Сделать сворачивающиеся и разворачивающиеся подробности
   - Добавьте поле с целью ЛР и подсветить слова: знакомство, HTML, CSS
   - Скопируйте для калькулятора стилистику веб-ресурса, которая не будет повторяться с остальными студентами
